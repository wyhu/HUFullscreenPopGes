# 使用方法

###1.只需添加2的文件，不需要任何设置，所有导航控制器将能够使用流行的手势自动全屏。


###2.要禁用导航控制器的这个弹出姿态：


    navigationController.fd_fullscreenPopGestureRecognizer.enabled = NO;

###3.要禁用视图控制器的这个弹出姿态：

    viewController.fd_interactivePopDisabled = YES;
